Пошаговая инструкция

01
telegram

Скопируйте ключ

  1. Откройте Telegram-бот и перейдите в меню.
  2. Нажмите кнопку 🔑 Обновить или введите /start.
  3. В сообщении бота найдите 🔗 Твоя ссылка для подключения.
  4. Нажмите на ключ, чтобы скопировать его в буфер обмена.
02
happ proxy

Скачайте приложение

Установите приложение Happ / Happ Proxy для своей платформы:

🍏 iOS App Store ↗ 🤖 Android Google Play ↗ 💻 Windows Скачать .exe ↗
03
импорт

Вставьте ключ

  1. Откройте скачанное приложение.
  2. Найдите кнопку Добавить профиль — обычно это + в правом верхнем углу.
  3. Выберите опцию Вставить из буфера обмена (Import from Clipboard).
04
подключение

Подключитесь

  1. Выберите только что добавленный профиль в списке.
  2. Нажмите большую кнопку Подключить.
  3. При первом подключении телефон может запросить разрешение на добавление VPN-конфигурации — нажмите Разрешить (Allow).
🎉 Готово! Значок VPN должен появиться в верхней панели вашего экрана.
// 02

Возможные проблемы

› Не подключается
Выключите VPN, обновите подписку в боте, скопируйте новый ключ и добавьте его заново — предварительно удалив старый профиль.
› Нет интернета после подключения
Попробуйте переключиться между мобильным интернетом и Wi-Fi, либо перезагрузите телефон.
› Ничего не помогает
Напишите в поддержку бота через команду /sup в меню — опишите проблему, и мы поможем.
