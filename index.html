Инструкция по подключению

1
Скопируйте ключ
  1. Откройте меню бота в Telegram.
  2. Нажмите кнопку 🔑 Обновить (или /start).
  3. В сообщении бота найдите 🔗 Твоя ссылка для подключения.
  4. Нажмите на ключ (ссылку https://...), чтобы скопировать его в буфер обмена.
2
Скачайте приложение

Установите клиентское приложение Happ/Happ Proxy:

🍏 iOS 🤖 Android 💻 Windows
3
Вставьте ключ
  1. Откройте скачанное приложение.
  2. В приложении найдите кнопку Добавить профиль (обычно это + в правом верхнем углу).
  3. Выберите опцию Вставить из буфера обмена (Import from Clipboard).
4
Подключитесь
  1. Выберите добавленный профиль в списке.
  2. Нажмите большую кнопку Подключить.
  3. При первом подключении система попросит разрешение на добавление туннеля — нажмите Разрешить (Allow).
🎉 Готово! Значок VPN должен появиться в верхней панели вашего устройства.
🛠 Возможные проблемы
  • Не подключается: Выключите VPN, обновите подписку в боте, скопируйте новый ключ и добавьте его заново (удалив старый).
  • Нет интернета: Попробуйте переключиться между мобильным интернетом и Wi-Fi, либо перезагрузите устройство.
  • Если ничего не помогает, напишите в поддержку бота через кнопку в меню /sup.
